# parser-subscribers-telegram
Parsing of subscribers of the channel telegram. 
To run the script, you need the:
<ul>
<li>installed <a href="https://github.com/LonamiWebs/Telethon">Telethon</a> library</li>
<li>get api_id and api_hash on the website <a href="https://my.telegram.org/">Telegram</a></li>
  <li><b>Attention:</b> you <b>must be</b> a channel administrator</li>
</ul>
# Usage
Add api_id and api_hash to the script and run:
<code>python3 main.py</code>
